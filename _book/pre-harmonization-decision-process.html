3 Pre-harmonization decision process

3.0.1 Initial dataset

At the start of the preharmonization process the WQP dataset contains 15.19 million rows.


3.0.2 Remove duplicates

The first filtering step is to remove any duplicated records present in the dataset. This results in dropping 3.01 million rows for a record count of 12.18 million.


3.0.3 Missing results

Next missing results are dropped from the dataset. 503.21 thousand rows are dropped, resulting in a final count of 11.68 million.


3.0.4 Filter status

We next filter the ResultStatusIdentifier column to include only the following statuses:

  • "Accepted"
  • "Final"
  • "Historical"
  • "Validated"
  • "Preliminary"
  • NA

44.82 thousand rows are dropped from the dataset leaving it with 11.63 million remaining.


3.0.5 Filter media

Next the ActivityMediaSubdivisionName is filtered to only include only the following media:

  • "Surface Water"
  • "Water"
  • "Estuary"
  • NA

11.95 thousand rows are dropped and 11.62 million remain.


3.0.5.1 Location type

The final step in pre-harmonization is filtering out any ResolvedMonitoringLocationTypeName that is not "Estuary", "Lake, Reservoir, Impoundment", or "Stream".

After dropping 0 thousand rows the pre-harmonization dataset is complete with 11.62 million.
